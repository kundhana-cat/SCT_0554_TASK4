# Traffic Accident Data Analysis

This project analyzes traffic accident data to identify patterns based on
road conditions, weather, and time of day. Visualizations are used to detect
accident hotspots and major contributing factors.

## Features
- Time-based accident analysis
- Weather & road condition insights
- Accident hotspot visualization
- Clean and simple plots

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python analysis.py

## Dataset
Accident dataset containing:
- Time
- Weather condition
- Road condition
- Latitude & Longitude
